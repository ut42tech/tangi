# Tangi — プロジェクト概要

## 一言で言うと

研究室にある特注のタンジブルユーザインタフェース（TUI）ハードウェア（5台）に対して、Web 開発者が `npm install` するだけでマーカー検出アプリケーションを作れるようにする OSS の TypeScript SDK。

## スコープ

**SDK が担うこと**

- ディスプレイ表面に置かれたマーカー（QR コード・ArUco マーカー）の検出
- 検出されたマーカーの位置・回転・ID のリアルタイム取得
- 複数台の TUI デバイス間でのマーカー状態の同期
- カメラ座標 → ディスプレイ座標へのキャリブレーション

**SDK が担わないこと**

- タッチ操作 — ブラウザ標準の `PointerEvent` / `TouchEvent` で十分
- UI コンポーネント — 通常の React アプリとして構築する
- アプリケーションロジック — SDK はインフラ層に徹する

## 利用イメージ

```
普通の React アプリ（タッチ対応の Web アプリ）
    + @tangi/core（マーカー検出 SDK）
    = TUI アプリケーション
```

例：受付システムなら、画面上のボタンをタッチで押す操作は通常の React の範疇。SDK が担うのは「名刺に貼った QR コードをディスプレイ上に置くと、そのユーザーが認識される」という部分のみ。

## ターゲットユーザー

- 一般の Web 開発者
- 研究機関（HCI / TUI 分野）
- 教育機関

研究機関がターゲットに含まれるため、**クラウド非依存・セルフホスト可能**が必須要件。

## パッケージ構成

| パッケージ              | 役割                         |
| ----------------------- | ---------------------------- |
| `@tangi/core`           | クライアント SDK 本体        |
| `@tangi/server`         | マルチデバイス同期サーバー   |
| `@tangi/detector-qr`    | QR コード検出プラグイン      |
| `@tangi/detector-aruco` | ArUco マーカー検出プラグイン |

## ステータス

設計完了・実装初期段階。最優先タスクは、超音波カメラのモノクロ映像で qr-scanner / js-aruco2 が動作するかの実機検証。
