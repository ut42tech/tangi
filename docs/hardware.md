# Hardware

研究室に特注で製作されたテーブルトップ型 TUI デバイスが **5 台**ある。本 SDK はこのハードウェアを統合的に扱うことを目的とする。

## コンポーネント構成

| コンポーネント            | 仕様                                                        |
| ------------------------- | ----------------------------------------------------------- |
| LCD ディスプレイ          | 光沢パネル、21〜24 インチ、16:9、バックライト内蔵、水平設置 |
| 超音波モノクロ Web カメラ | LCD パネル背面に取付。ディスプレイ表面のマーカーを検出      |
| タッチパネル              | 静電容量方式                                                |
| 筐体                      | 箱型、テーブルトップ（水平）設置                            |

## 外部接続

| 端子     | 用途         | デバイス認識 |
| -------- | ------------ | ------------ |
| HDMI     | 映像入力     | —            |
| USB（1） | Web カメラ   | UVC デバイス |
| USB（2） | タッチパネル | HID デバイス |

## 重要な特徴

- 専用ドライバ・専用ソフトウェアは一切ない。各コンポーネントは OS 標準の汎用ドライバで動作する
- つまり「ハードウェアの集合体」でしかなく、これを「1 つのタンジブルシステム」として統合するソフトウェアが本 SDK の役割
- 超音波カメラは可視光ではなくモノクロ映像を出力する。可視光カメラだとディスプレイの発光と干渉するため、超音波でそれを回避している
- ディスプレイは水平設置で使用する。マーカー検出はディスプレイ表面に直接マーカーを配置して行う
